# Project Workflow

## Overview

The Human Development Index (HDI) Prediction System is developed using Machine Learning and Flask. The project follows a structured workflow beginning with environment setup and ending with deployment as a web application.

---

## Workflow

### Step 1: Environment Setup
- Install Python
- Create Virtual Environment
- Install required packages
- Configure VS Code
- Create GitHub repository

### Step 2: Import Required Libraries
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Flask
- Pickle

### Step 3: Dataset Download and Understanding
- Download HDI dataset
- Load dataset using Pandas
- Explore features
- Analyze data types
- Perform data visualization

### Step 4: Data Preprocessing
- Handle missing values
- Select dependent and independent variables
- Perform label encoding
- Prepare clean dataset

### Step 5: Train-Test Split
- Divide dataset into training and testing datasets

### Step 6: Model Training
- Train Linear Regression model
- Predict HDI score
- Evaluate model performance

### Step 7: Save Model
- Save trained model using Pickle
- Store model for future prediction

### Step 8: Flask Web Application
- Develop Flask backend
- Create HTML frontend
- Connect model with Flask
- Display prediction results

---

## Conclusion

The project follows a complete Machine Learning lifecycle from data collection to deployment, ensuring a structured, maintainable, and scalable HDI prediction system.
